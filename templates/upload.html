CSV Format Requirements

Your CSV file should have the following columns:

  • url - The URL to monitor (with or without http/https)
  • group_name - The group this URL belongs to
  • countryCode - Country code for cookie header (optional)

Example:

url,group_name,countryCode
https://www.google.com,Search Engines,US
github.com,Development,US
stackoverflow.com,Development,UK
